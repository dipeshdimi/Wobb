## Requirements

Ensure you have the following installed:

- **Node.js** (v14.18.0+)
- **npm** or **yarn**

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dipeshdimi/BitScale.git
   ```

2. Navigate to the project directory:
   ```bash
   cd BitScale
   ```
   
3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm run dev
   ```
